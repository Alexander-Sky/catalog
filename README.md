# Blog Lorem

Простой блог-сервер на Python с использованием Bootstrap.

## Технологии
- Python (встроенный HTTP-сервер)
- Bootstrap 5
- HTML/CSS

## Установка и запуск

```bash
# Перейти в папку проекта
cd catalog

# Запустить сервер
python blog_server.py
Открыть в браузере: http://localhost:8080

Структура проекта
text
catalog/
├── blog_server.py      # основной сервер
├── templates/          # HTML-шаблоны
│   ├── index.html      # главная страница
│   ├── contacts.html   # страница с формой обратной связи
│   ├── about.html      # информация о проекте
│   └── 404.html        # страница ошибки
├── static/
│   └── css/
│       └── style.css   # пользовательские стили
├── .gitignore
├── README.md
└── pyproject.toml
Функциональность
GET-запросы — возвращают HTML-страницы

POST-запросы — форма обратной связи (данные выводятся в консоль сервера)

Обработка ошибок — страница 404 для несуществующих маршрутов
